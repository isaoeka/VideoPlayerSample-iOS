# VideoPlayerSample-iOS
The video player sample for iOS App.

## Usage

```
$ make install
```

Next, open VideoPlayerSample-iOS.xcodeproj and run an application.

## Requirements
Xcode 10.0
Swift 4.2
Carthage 0.31.0
